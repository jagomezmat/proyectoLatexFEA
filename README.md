# proyectoLatexFEA
Latex del proyecto final de Física Experimental Avanzada

Realizar la actualización de la rama remota de su compañero directamente desde la consola de VS Code
# 1. Descargar las referencias remotas
git fetch origin

# 2. Cambiar temporalmente a la rama de Jhon
git checkout ramaJhon

# 3. Fusionar el contenido de main en ramaJhon
git merge main

# 4. Enviar los cambios actualizados a GitHub
git push origin ramaJhon

# 5. Regresar a su rama principal de trabajo
git checkout main
